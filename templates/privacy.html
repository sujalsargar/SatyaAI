Privacy & Data

SatyaAI processes uploads and user inputs only for verification purposes. Uploaded files are stored temporarily in the server upload folder and removed after processing (unless you keep them in history).

What we store

  • Analysis results (status, confidence, reasoning, sources, risk score)
  • Small text snippet of the input (truncated)
  • User metadata for account and history management

What we don't share

  • We never publish your uploaded media or private data.
  • OpenAI / LLM calls use your configured API key; usage occurs on the server and costs are billed to the key owner.

If you want data deleted, use the Dashboard to remove history items or contact the administrator.
